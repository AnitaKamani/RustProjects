# Reverser Machine
Problem: https://quera.org/problemset/3405

input: n lines of numbers until it recieves a zero
 

ouput: the numbers in reversed order in n lines with no leading 0z


e.g.: 

    input:      3
                4
                7
                4
                9
                0


    output:     9
                4
                7
                4
                3
